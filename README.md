# todoX - 代辦事項 X

[Demo](https://todox-74ff1.web.app/)

此專案能為每一個主題紀錄代辦事項，主題及代辦事項皆能新增、刪除及修改，並提供將資料存在資料庫。

## 功能列表

- 使用者能註冊、登入及登出
- 新增、刪除及修改不同主題的標題及內容
- 新增、刪除及修改不同代辦事項的標題及內容
- 主題依照新增日期或修改日期排序
- 代辦事項可以自訂排序
